## SYNOPSIS

Compares the actual value to a boolean $true.
It converts input values to boolean, and will fail for any value is not $true, or truthy.

## SYNTAX

```powershell
Should-BeTruthy [[-Actual] <Object>] [[-Because] <String>] [<CommonParameters>]
```

## DESCRIPTION

\{\{ Fill in the Description \}\}

## EXAMPLES

### EXAMPLE 1

```powershell
$true | Should-BeTruthy
1 | Should-BeTruthy
Get-Process | Should-BeTruthy
```

This assertion will pass.

### EXAMPLE 2

```powershell
$false | Should-BeTruthy
$null | Should-BeTruthy
$() | Should-BeTruthy
@() | Should-BeTruthy
0 | Should-BeTruthy
```

All of these assertions will fail, because the actual value is not $true or truthy.
